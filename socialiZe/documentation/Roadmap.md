# Roadmap

## Planned Features

1. **Show Both Camera and Preview**
    - The application will have the ability to display both the camera feed and a preview of the recorded video.

2. **Custom FPS**
    - Users will be able to set a custom Frames Per Second (FPS) value for the video recording.

3. **Metadata Generation**
    - The application will generate metadata for each recording. This metadata will include all the parameters used for the recording and will be stored in a JSON format.

4. **Recording Organization**
    - Each recording will be stored in a new folder (along with relevant metadata and auxillary files) to keep the project organized.

5. **Automatic Dot Reset**
    - When a recording is started, the dot will automatically reset to its initial position.

The order of implementation or the features themselves may change as the project progresses.
