Welcome to the socialiZe documentation! This guide will walk you through the steps necessary to get socialiZe up and running on your system.

## What is socialiZe?

socialiZe is a web application designed for behavioral neuroscientists to analyze social behavior in zebrafish. It provides tools to manipulate the visual environment and analyze zebrafish responses via a live video stream.

## Who is this documentation for?

This documentation is intended for researchers, developers, or anyone interested in analyzing zebrafish behavior using socialiZe. Whether you're new to the tool or an experienced user, these guides will provide the information you need.

## How to use this documentation

This "Getting Started" guide will walk you through the following steps:

1. **Prerequisites**: Installing all necessary software dependencies.
2. **Installation**: Setting up the core components of socialiZe.

Once you've completed the installation process, explore the rest of this documentation for detailed usage instructions, advanced configuration options, and more.
